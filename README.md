🚀 AtliQ Hospitality Revenue Analytics Dashboard | Power BI

I'm excited to share my latest Power BI project: AtliQ Hospitality Revenue Analytics Dashboard, developed using a real-world hospitality dataset from the Codebasics Hospitality Challenge.

🏨 Project Overview

The objective of this project was to analyze hotel booking and revenue data for AtliQ Grands, a luxury hotel chain facing declining market share due to increasing competition. The goal was to transform raw business data into meaningful insights that support strategic decision-making through an interactive Power BI dashboard.
